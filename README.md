## 二、视频选题简介
**视频题目：** 具身智能，让AI真正走进物理世界
**选题方向：** 什么是具身智能？与传统AI有何不同？
**视频时长：** 3分12秒

### 内容大纲
1.  **开头（约30秒）**：
    以ChatGPT、Siri等大家熟悉的AI为引，提出问题：“为什么这些AI‘很聪明’，却不能像机器人一样开门、倒水？”引出“具身智能”的概念。
2.  **主体（约2分钟）**：
    - 对比讲解：传统AI vs 具身智能
    - 核心技术拆解：感知、规划、控制、仿真
    - 案例展示：波士顿动力、Figure AI、宇树机器人等
3.  **结尾（约30秒）**：
    总结具身智能如何打通“数字大脑”和“物理身体”，并展望未来可能的应用场景。
